STEP 1: BUAT REPOSITORY DI GITHUB
========================================

1. Buka browser, login ke GitHub
2. Buka: https://github.com/new
3. Isi form:
   - Repository name: animestream (atau nama lain)
   - Description: Website streaming anime
   - Pilih: Public atau Private (terserah)
   - JANGAN centang "Initialize with README"
4. Klik "Create repository"

STEP 2: COPY URL REPOSITORY
========================================

Setelah repository dibuat, GitHub akan tampilkan URL seperti:
https://github.com/USERNAME_KAMU/animestream.git

Copy URL tersebut!

STEP 3: SET REMOTE URL
========================================

Buka terminal/command prompt di folder project, lalu jalankan:

git remote set-url origin https://github.com/USERNAME_KAMU/animestream.git

(Ganti USERNAME_KAMU dengan username GitHub kamu)

STEP 4: PUSH KE GITHUB
========================================

git push -u origin main

Kalau diminta login:
- Username: username GitHub kamu
- Password: JANGAN pakai password biasa!
  Pakai Personal Access Token (PAT)
  
  Cara buat PAT:
  1. GitHub â†’ Settings â†’ Developer settings
  2. Personal access tokens â†’ Tokens (classic)
  3. Generate new token (classic)
  4. Centang: repo
  5. Generate token
  6. Copy token (simpan baik-baik!)
  7. Pakai token ini sebagai password

STEP 5: VERIFIKASI
========================================

Buka repository di GitHub, refresh halaman.
Semua file seharusnya sudah terupload!
